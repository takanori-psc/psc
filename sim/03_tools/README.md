# tools

Common utilities for PSC simulation.

## Contents

- `psc_plot.py`: visualization tool
- `run_sim.sh`: simulation runner script

## Scope

These tools are shared across:

- basic simulations
- advanced experiments

## Notes

This directory contains reusable utilities.  
It is not tied to a specific simulation version.

## Usage

Run the advanced policy-aware routing simulation:

```bash
chmod +x sim/03_tools/run_sim.sh
./sim/03_tools/run_sim.sh
```
