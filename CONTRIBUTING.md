# Contributing to PSC

Thank you for your interest in PSC (Photon System Controller).

## 🧠 About this project

PSC is a research-oriented project exploring a fabric-centric computer architecture.

This project is currently under active design and prototyping.

## 🚧 Current Status

- Early stage (v0.1)
- Rapid changes expected
- Core architecture is still evolving

## 🤝 How to Contribute

You are welcome to:

- Open issues for questions or discussions
- Suggest improvements to documents
- Propose architectural ideas
- Submit pull requests (PRs)

## 📌 Guidelines

- Keep discussions constructive
- Prefer clarity over complexity
- Respect the design philosophy of PSC

## ⚠ Important Note

PSC is a highly experimental project.

Breaking changes may occur frequently.

## 💬 Discussions

Please use GitHub Discussions for:

- Questions
- Ideas
- Architecture discussions

Thank you for your support!
