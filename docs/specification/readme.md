# PSC Specification

This directory contains PSC specification documents.

このディレクトリには PSC の仕様書が含まれています。

---

## Directory Structure

| Directory | Description (EN)                                            | 説明 (JA)                  |
|-----------|-------------------------------------------------------------|----------------------------|
| published | Specifications that are already published in the repository | リポジトリで公開済みの仕様 |
| draft     | Work-in-progress specifications                             | 作成中の仕様               |
| archive   | Deprecated or historical specifications                     | 廃止または旧仕様           |

---

## Example

Example file naming convention:
```
psc_fabric_state_model_v0.1_en.md
psc_fabric_state_model_v0.1_ja.md
```
Meaning:

| Element            | EN                 | JA             |
|--------------------|--------------------|----------------|
| psc                | Project name       | プロジェクト名 |
| fabric_state_model | Specification name | 仕様名         |
| v0.1               | Version            | バージョン     |
| en                 | English version    | 英語版         |
| ja                 | Japanese version   | 日本語版       |
