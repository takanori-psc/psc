PSC Address Format v0.1 は、
固定長64ビットの階層型アドレス構造を採用する。

このアドレスは以下のフィールドで構成される。

Fabric ID   : 16ビット
Cluster ID  : 16ビット
Node ID     : 24ビット
Port ID     : 8ビット

この形式は、
効率的なハードウェア実装、
明確な階層ルーティング、
および小規模ローカルPSCシステムから
大規模分散リージョナル構成まで対応可能な
拡張性を目的として設計される。

この64ビットPSCアドレスは、
PSC Fabric 内部でルーティングおよび転送実行に使用される
ネイティブファブリックアドレスとして定義される。

上位の論理名、サービス識別子、
および広域スケールのオブジェクト参照は、
Resolver などの上位解決層で扱うものとし、
ネイティブ転送アドレスへ直接埋め込むことを必須としない。
